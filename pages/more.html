### Debugging Because Framer runs in the browser you can use all the debugging tools, like the console, timeline, profiler and debugger. Framer will alert you when a Javascript error occurred.
Key Description
Escape (toggle) Switch between normal & debug view of all layers. Useful to inspect structure and positions.
Shift (hold) While shift is pressed, Framer will delay the time by 25x. This will slow down animations and timers like utils.delay and utils.interval. Useful to inspect animations carefully.
### Questions There is an active and helpful Framer Community on Facebook.
  • Can I use Framer with Sketch?
    • Not yet, but we're working with the Sketch team to get it in Framer Generator.
  • Can I use Framer on Windows?
    • Yes. The core of Framer is a Javascript library that works on all platforms.
  • Why does performance on mobile seem worse?
    • Mobile browsers have a 500ms click delay build in, to distinguish a click event from a hover or right click. Use a "touchstart" or "touchend" event instead of a "click" event.
