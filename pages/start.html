{% extends "base.html" %} {% block sidebarclass %}start{% endblock %} {% block content %} {% filter markdown %} ### Framer.js Framer consists of 2 parts – the Framework and Generator. The core of Framer is a Javascript Framework that enables you to define interactions and animations in a simple, readable way. It's a small framer.js file that makes creating realistic prototypes a breeze – complete with animations, spring physics and 3D effects. Framer Generator optimizes the flow of going from static mockups to interactive prototypes. It's a desktop application that extracts the resources and layer hierarchy out of Photoshop files (Sketch coming soon) and renders them to a Framer project folder, where you can easily add interaction and animation to the layers.
### Installation To get started, download Framer and unzip it. Here's what you get:
  • Project folder
    • app.js
    • index.html
    • Framer folder
    • Images folder 
  • Framer Generator.app
Open index.html to view the default prototype and you'll see the logo bounce on a click. To add interactions and animations, start editing app.js (in your code-editor). Try it by adding a state to imageLayer: 
fifth: { y:200, scale:0.8, rotationZ:200, blur:2 }
The imageLayer will now move up, scale, rotate and blur on the fourth click. Framer works, congrats! Note: Framer runs best in WebKit, so please use Safari. Chrome works too, but has been rocky since they switched to Blink.
#### JavaScript Framer makes it easier to prototype animations and interactions, but you will benefit from understanding a bit of JavaScript. If you don't know JavaScript or are just beginning, don't worry. There are many great resources to get started and while it might seem hard at first you'll see that you'll get better quickly. It's also good to know that JavaScript is worth learning apart from Framer, as you can do web, application, and backend development with it.

### Generator The Framer Generator.app takes static mockups from Photoshop (Sketch coming soon) and turns them into ready-to-go Framer projects. From these apps, it extracts resources and folder hierarchy so you can start adding interactions and animations immediately. With a Photoshop file active, open Framer Generator and click Import. Only layer groups are imported; single layers are ignored. The hierarchy of your groups determine the layer hierarchy in Framer. After importing you will be presented with a folder containing your Framer project. Open index.html to view the project, and start editing app.js to add behavior. You can safely move things around in Photoshop and import again. Framer Generator will update the images and hierarchy, but leave your code safe. The basics for converting Photoshop files:
  • Every layer group will become a Framer Layer, with an image of all the layers
  • By default your layers are imported under a myLayers object
  • Access any layer by its group name: myLayers.GroupName or myLayers["Group Name"]. For example, if you had a layer group named 'avatar' in Photoshop, you would access it in Framer with myLayers.avatar
  • Framer respects the group hierarchy, so a group in a group will become a subLayer
  • Groups with vector masks will become layers with clip enabled
  • Group names should be unique. If they're not, Framer will rename them on export
