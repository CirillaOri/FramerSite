Framer.js

A JavaScript framework for rapid prototyping. With Filters, Spring Physics and 3D effects.

Download GitHub

Framer consists of two parts: Framer.js and Framer Generator. Framer.js is a JavaScript Framework that enables you to define interactions and animations in a simple, readable way.

Framer Generator is a desktop app that extracts layers out of Sketch or Photoshop files and exports them to a Framer project folder, where you can easily add interaction and animation to the layers.

{% filter markdown %} ### Installation To get started, download Framer and unzip it. Open index.html to view the default prototype and you'll see the logo bounce on a click. To add interactions and animations, start editing app.js (in your code-editor). Try it by adding a state to imageLayer: 
fifth: { y:200, scale:0.8, rotationZ:200, blur:2 }
The imageLayer will now move up, scale, rotate and blur on the fourth click. Framer works, congrats! Note: Framer runs best in WebKit, so please use Safari. Chrome works too, but has been rocky since they switched to Blink.
#### JavaScript Framer makes it easier to prototype animations and interactions, but you will benefit from understanding a bit of JavaScript. If you don't know JavaScript or are just beginning, don't worry. There are many great resources to get started. It's also good to know that JavaScript is worth learning apart from Framer, as you can do web, application, and backend development with it.

### Generator The Framer Generator.app takes static mockups from Sketch or Photoshop and turns them into ready-to-go Framer projects. From these apps, it extracts resources and folder hierarchy so you can start adding interactions and animations immediately. With a Sketch or Photoshop file active, open Framer Generator and click Import. Only layer groups are imported; single layers are ignored. The hierarchy of your groups determine the layer hierarchy in Framer. After importing you will be presented with a folder containing your Framer project. Open index.html to view the project, and start editing app.js to add behavior. You can safely move things around in Sketch or Photoshop and import again. Framer Generator will update the images and hierarchy, but leave your code safe. The basics for converting Sketch and Photoshop files:
  • Every layer group will become a Framer Layer, with an image of all the layers
  • By default your layers are imported under a myLayers object
  • Access any layer by its group name: myLayers.GroupName or myLayers["Group Name"]. For example, if you had a layer group named 'avatar' in Photoshop, you would access it in Framer with myLayers.avatar
  • Framer respects the group hierarchy, so a group in a group will become a subLayer
  • Groups with vector masks will become layers with clip enabled
  • Group names should be unique. If they're not, Framer will rename them on export
