Property Description Examples
Basic
id layer id (readonly)
name layer name
properties all current properties for this layer
Geometry
x minX, midX, maxX x position 1 2
y minY, midY, maxY y position
z z position
width width
height height
frame object with x, y, width and height values
screenFrame() get this layers absolute frame on the screen
contentFrame() the frame for the total size of all the sub layers combined 1
centerFrame() the centered frame for this layer
center() centerX(), centerY() center this layer
pixelAlign() round the x and y values to whole pixels
Visual
backgroundColor set html background color
visible toggles layers visibility
opacity opacity between 0 and 1
clip toggles wether layer should clip sub layers
rotation rotationX, rotationY, rotationZ rotation in degrees
scale scaleX, scaleY, scaleZ scale between 0 and 1
originX, originY origin of scale and/or rotation transformation
Images
image set the image url for this layer
Hierarchy
superLayer the layer that contains this layer
subLayers the layers contained by this layer
subLayersByName("name") returns array of sub layers by name
addSubLayer(layer) add a sub layer to this layer
removeSubLayer(layer) remove a sub layer from this layers sub layers
siblingLayers the sub layers of this layers superlayer
Ordering
index this layers layering position index
placeBefore(layer) place this layer on top of other layer
placeBehind(layer) place this layer behind another layer
bringToFront() place this layer in front of all sibling layers
sendToBack() place this layer behind all sibling layers
HTML
style css properties for this layer
html html contents for this layer
destroy() destroy this layer
Filters
blur gaussian blur in pixels
brightness brightness in percent
saturate saturation in percent
hueRotate hue rotation in degrees
contrast contrast in percent
invert color inversion in percent
grayscale grayscale in percent
sepia sepia in percent
Animation

animateStop() stop all animations on this layer
Scrolling

scrollFrame current scroll position for layer
Events

### Animations
Property Description Examples
layer layer that is animated
properties properties being animated (x, y, width, height, opacity, scale, rotation, filters)
curve curve used for animation
  • linear, ease-in, ease-out
  • bezier-curve(a, b, c, d)
  • spring(tension, friction, velocity)
    Tension is the stiffness of the spring, friction the amount of effort it takes to move the object and velocity the initial velocity of the object.
time time to animate in seconds (ignored for springs)
start() start the animation
stop() stop the animation
reverse() returns the opposite animation to this one
on(event, function) listen to events: end
### Utilities
Function Description Examples
cycle([1, 2, ...]) Returns a function that returns the next value every time you call it.
randomColor(alpha) Returns a random color string.
delay(time, function) Executes the function after time in ms. Holding shift will slow this down 25x.
interval(time, function) Executes the function every x ms.
throttle(time, function) Call function at most once every x ms.
debounce(time, function) Call function only when being stopped calling for x ms.
convertPoint(point, layerA, layerB) Converts a point location from one layers coordinate system to another.
max([..]) Returns the largest value in an array.
min([..]) Returns the smallest value in an array.
sum([..]) Returns the sum of an array.
uuid() Returns a random hex uuid string.
round(number, decimals) Rounds a number to x decimals.
isWebKit() Returns wether this is a webkit browser.
isMobile() Returns wether this is a mobile device.
isTouch() Returns wether this is a touch device.
### Defaults
Property Description Examples
timeSpeedFactor Current speed of time. Used to slow down animations and timers. Default value is 1.
roundingDecimals Rounding setting when generating css values. Default value is 5.
animationPrecision Number of css keyframes per second to generate for animations. Default value is 60.
animationDebug Global setting to debug all animations. Default value is false.
animationProfile Global setting to profile all animations. Default value is false.
