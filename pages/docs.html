Property Description
Geometry
x minX, midX, maxX x position
y minY, midY, maxY y position
width width
height height
frame this frame x, y, width and height values
screenFrame get this views absolute frame on the screen
contentFrame() the frame for the total size of all subviews
centerFrame() the centered frame in this views superView
Visual
visible toggles views visibility
opacity opacity between 0 and 1
clip toggles wether view should clip subviews
rotation rotationX, rotationY, rotationZ rotation in degrees
scale scaleX, scaleY, scaleZ scale between 0 and 1
Hierarchy
superView the view that contains this view
subViews the views contained by this view
addSubView(view) add a subview to this view
removeSubView(view) remove a subview from this views subviews
siblingViews the subviews of this views superview
Ordering
index this views layering position index
placeBefore(view) place this view on top of other view
placeBehind(view) place this view behind another view
bringToFront() place this view in front of all sibling views
sendToBack() place this view behind all sibling views
HTML
style css properties for this layer
html html contents for this layer
destroy() destroy this view
Filters
blur gaussian blur in pixels
brightness brightness in percent
saturate saturation in percent
hueRotate hue rotation in degrees
contrast contrast in percent
invert color inversion in percent
grayscale grayscale in percent
sepia sepia in percent
Animation
animate({...}) create and start an Animation on this view, returns the Animation object
animateStop() stop all animations on this view
on(event, function) listen to events: click, mouseover, change:property, etc
### Animations
Property Description
view view that is animated
properties properties being animated (x, y, width, height, opacity, scale, rotation, filters)
curve curve used for animation
  • linear, ease-in, ease-out
  • bezier-curve(a, b, c, d)
  • spring(tension, friction, velocity)
    Tension is the stiffness of the spring, friction the amount of effort it takes to move the object and velocity the initial velocity of the object.
time time to animate in milliseconds (ignored for springs)
origin origin to transform from in percent "50% 50%"
start() start the animation
stop() stop the animation
reverse() returns the opposite animation to this one
on(event, function) listen to events: end
